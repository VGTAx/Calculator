# Calculator

Калькулятор — это простое веб-приложение, выполняющее базовые арифметические вычисления. Он поддерживает сложение, вычитание, умножение и деление, а также функции процента, квадрата числа, плюс/минус, обратной величины числа, квадратного корня. Калькулятор построен с использованием HTML, CSS и JavaScript.


<ul>

<b>Функции</b>:
	<li>Сложение: выполнение операций сложения между двумя числами.<br></li>
	<li>Вычитание: выполнение операций вычитания между двумя числами.<br></li>
	<li>Умножение: выполнение операций умножения между двумя числами.<br></li>
	<li>Деление: выполнение операций деления между двумя числами.<br></li>
	<li>Процент: вычисление процента от числа.<br></li>
	<li>Квадрат числа: вычисление квадрата числа.<br></li>
	<li>Обратная величина числа: вычисляет число, умножение на которое даёт единицу.<br></li>
	<li>Плюс/минус: меняет знак числа на противоположный.<br></li>
	<li>Квадратный корень: вычисление квадратного корня числа.<br></li>
</ul>



<ul>

<b>Используемые технологии</b>:
	<li>HTML<br></li>
	<li>CSS</li>
	<li>JavaScript</li>
</ul>



<b>Инструкции по установке и использованию:</b><br>

Чтобы использовать калькулятор, перейдите по ссылке https://vgtax.github.io/Calculator/ в веб-браузере. Введите первое число, выберите оператора, введите второе число и нажмите кнопку «Рассчитать». Результат будет отображен на экране.

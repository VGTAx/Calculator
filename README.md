# Calculator

The calculator is a simple web application that performs basic arithmetic calculations. It supports addition, subtraction, multiplication and division, as well as percent, square, plus/minus, reciprocal, and square root functions. The calculator is built using HTML, CSS and JavaScript.


<ul>

<b>Functions</b>:
<li>Addition: Performing addition operations between two numbers.<br></li>
<li>Subtraction: Perform subtraction operations between two numbers.<br></li>
<li>Multiplication: Performing multiplication operations between two numbers.<br></li>
<li>Division: Perform division operations between two numbers.<br></li>
<li>Percentage: Calculate the percentage of a number.<br></li>
<li>Squaring a number: Calculate the square of a number.<br></li>
<li>Reciprocal of a Number: Calculates a number that when multiplied by gives one.<br></li>
<li>Plus/minus: reverses the sign of a number.<br></li>
<li>Square Root: Calculate the square root of a number.<br></li>
</ul>



<ul>
	
<b>Technologies used</b>:
<li>HTML<br></li>
<li>CSS</li>
<li>JavaScript</li>
</ul>


<b>Installation and use instructions:</b><br>

To use the calculator, go to https://vgtax.github.io/Calculator/ in a web browser. Enter the first number, select an operator, enter the second number and click the "Calculate" button. The result will be displayed on the screen.
